Link Browser  
A modern, lightweight web browser built with JavaFX using FXGL Engine.  

Features  
- Tabbed browsing – Open multiple pages at once  
- Progress bar & status updates – See page load progress  
- Custom UI with JavaFX & TilesFX – Sleek, modern interface  
- CPU & Memory monitoring – Real-time performance tracking  
- Dark mode & customizable themes  
- Open-source & actively maintained  

Installation  
Prerequisites  
- Java 17+  
- JavaFX latest version  

Running the Browser  
1. Clone the repository  
   git clone https://github.com/Kobi401/LinkBrowser.git  
2. Navigate to the project directory  
   cd LinkBrowser  
3. Run using Gradle  
   ./gradlew run  
   Or using Maven  
   mvn javafx:run  

Dependencies  
Link Browser is built with:  
- JavaFX – UI framework  
- ControlsFX – Enhanced UI components  
- TilesFX – Beautiful dashboard tiles

### Link Browser  
A modern, lightweight web browser built with JavaFX using FXGL Engine.  

![Screenshot_20250321_181932](https://github.com/user-attachments/assets/944539c7-279b-431d-a922-7fcac6d7646b)

### Features  
- Tabbed browsing (Hover mouse over tabs to see memory usage) 
- Support for injecting JS Scripts  
- Auto Check for updates 
- Simple yet clean UI 
- Themes with dark-mode and custom themes [SOON]
- Downloading support
- Runs on Windows 10/11 & Linux (mainly tested on Arch Linux)
- Open-source & actively maintained  

### Installation  
Prerequisites  
- Java 17+  
- JavaFX latest version  

### Running the Browser  
1. Clone the repository  
   git clone https://github.com/Kobi401/LinkBrowser.git  
2. Navigate to the project directory  
   cd LinkBrowser  
3. Run using Gradle  
   ./gradlew run  
   Or using Maven  
   mvn javafx:run  

### Dependencies  
Link Browser is built with:  
- JavaFX – UI framework  
- ControlsFX – Enhanced UI components  
- TilesFX – Beautiful dashboard tiles
- FXGL – JavaFX Game Development Framework
